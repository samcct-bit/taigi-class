# Taigi Class (台語教室) ── AI 輔助台語教材生成與整理專案

本專案旨在透過 AI 與自動化工具，收集、整理台灣中小學本土語（台語）官方教材，並最終達成「以自然語言自動生成各類台語教材」的核心願景。

---

## 🎯 專案核心目標

### 階段一：教材收集、分類與分析 (當前任務)
1. **官方教材檢索**：自動或半自動搜尋並下載台灣中小學本土語（台語）的官方教材與資源。
2. **教材整理與分類**：將下載的教材依據學區、年級、難度、主題進行系統化分類。
3. **教材內容分析**：分析教材中的詞彙量、句型結構、拼音系統（台羅/白話字）及教學大綱，建立結構化的知識庫。

### 階段二：自然語言教材生成 (未來核心目標)
使用者只需在專案中使用自然語言（例如：「產生一張國小三年級、關於水果主題的台語考卷」），系統即可自動產出以下內容：
1. 📄 **考卷**：包含聽力、選擇、配對、翻譯等多元題型的 PDF 或 Word 考卷。
2. 📊 **簡報 (PPT/Slides)**：結構清晰、配有拼音與解釋的教學簡報。
3. 🎥 **影片**：自動生成包含台語配音、字幕（台羅/漢字對照）的教學或故事短片。
4. 🌐 **數位互動網站及程式**：包含台語小遊戲、互動式發音練習、配對遊戲等 Web 應用。
5. 📝 **測驗 (Quizzes)**：線上互動式測驗（如 Kahoot 格式、Google 表單或自建線上測驗）。

## 📂 預計目錄結構

為了達成上述目標，專案初期規劃如下目錄結構：

```text
taigi-class/
├── README.md               # 專案目標與說明文件
├── docs/                   # 相關研究與分析文件
├── src/                    # 核心程式碼 (下載器、分析器、生成器)
│   ├── scrapers/           # 負責爬取與下載官方教材的腳本
│   ├── analyzer/           # 教材內容分析與資料結構化工具
│   └── generators/         # 各類教材生成器 (考卷、簡報、互動網頁等)
├── data/                   # 數據與教材存放目錄
│   ├── raw/                # 原始下載的官方教材 (PDF, 音檔, 影片等)
│   ├── processed/          # 整理、分類並結構化後的教材资料
│   └── templates/          # 用於生成教材的簡報、考卷、網頁模板
└── tests/                  # 測試用例
```
